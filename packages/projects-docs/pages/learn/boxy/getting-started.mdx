# Getting Started
All AI tools are available for Pro users. Simply grant permissions in team settings. 

<!-- todo: insert image -->

In order for the changes to take effect, you will need to restart your VMs. You can do this by selecting `Restart Branch` in the main menu. If you are in a cloud sandbox, the option will read `Restart Sandbox`


## Boxy for Sandboxes
Boxy is available only on repositories and cloud sandboxes. If you want the support of Boxy in your browser sandbox, you can easily convert your sandbox using the options in the toolbar.
